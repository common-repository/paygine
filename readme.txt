Интеграция платежной системы Method в Wordpress.

== Description ==

Плагин «Method» – платежное решение для сайтов на WooCommerce:
- Включает 9 встроенных способов приема платежей
- Подходит для юридических лиц и ИП
- Быстрое поступление денег на счет компании
- Полное соответствие 54-ФЗ
- Бесплатное подключение

== Настройка плагина ==

Чтобы принимать платежи через плагин, подайте заявку на подключение к Method на сайте methodpay.ru. С вами свяжется сотрудник компании и выдаст доступ на подключение.

Поддержка передачи чека
    1. Настройте быструю передачу чеков в налоговую из Method согласно 54-ФЗ.
    2. Передавайте чеки через партнеров (облачные кассы «АТОЛ Онлайн», Orange Data, «Бизнес.ру», Kit online, Cloud Kassir).

== Тарифы ==

Комиссия за прием платежей — от 1%. Комиссия рассчитывается индивидуально для каждого бизнеса. Для корректной оценки тарифа необходимо предоставить информацию по виду деятельности компании, среднему чеку и оборотам в месяц.

== Все возможности Method ==

После подключения Method доступны 9 способов приема платежей:
- Банковские карты
- СБП
- Яндекс Pay
- Mir Pay
- Sber Pay
- МТС Pay
- Плайт
- Подели
- FINBOX

+ Личный кабинет, в котором вы сможете видеть все операции, возвращать платежи, выставлять и отправлять счета и обратиться в службу поддержки Method (служба поддержки Method работает 24/7).
